### 1. 企業與產品研究 ([Companies/README.md](Companies/README.md))
紀錄各家資安廠商的核心亮點與主打產品，並統整相關安全解決方案：
- **端點與威脅偵測 (EDR/MDR/XDR)**：奧義智慧 (XASM)、TrendAI (SecOps)、TEAMT5 (ThreatVision/Sonar)
- **人工智慧安全 (AI Security / API Security)**：Akamai AI、Radware、零壹科技 (預防 prompt injection)
- **自動化實做與工控安全 (OT)**：LUCENT SKY (自動化弱點修正)、TXOne Networks (工控資安)

### 2. 議程筆記 ([Sessions/Session.md](Sessions/Session.md))
記錄參與的展會資安議程與技術分享。
- **精選議程**：*全面防護 AI* (講者：Richard Lee, CrowdStrike 北亞區技術顧問)
- **探討議題**：
  - 員工端點控管：防範 AI 資料外洩 (Prevent AI Data Leaks)、影子 AI 監測 (Uncover Shadow AI)
  - 企業自研 AI 保護：防止提示詞注入攻擊 (Stop Prompt Injection Attacks)
